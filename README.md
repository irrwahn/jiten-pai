# Jiten-pai

A Gjiten-lookalike Japanese dictionary written in Python.

## Requirements

Running `jiten-pai.py` requires Python version >= 3.7 installed,
plus the PyQt5 python bindings for Qt5.

Tested with Python 3.7.3, PyQt5 5.11.3.
